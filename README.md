# music-app
Music store where you can search for bands and music

# Tech stack
 - Java 17
 - Maven
 - Mongo DB
 - Spring Cache

# Build project
 - Move to the project path
 - ./mvnw clean install

# Run by Maven
 - Move to the project path
 - ./mvnw spring-boot:run

# Run by Docker 
- Move to the project path
- docker compose up

# Swagger
 - http://localhost:8081/swagger-ui/index.html

# Open Api Doc
 - http://localhost:8081/v3/api-docs
